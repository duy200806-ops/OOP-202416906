LAB 4:

Câu 1: Lớp nào nên triển khai (implement) interface Comparable?
Trả lời: Lớp Media nên triển khai (implement) interface Comparable<Media>. Vì danh sách giỏ hàng (Cart) hay cửa hàng (Store) chứa các đối tượng kiểu Media, việc implement Comparable ở lớp cha Media giúp ta có thể so sánh và sắp xếp danh sách chứa bất kỳ loại Media nào (Book, CD, DVD) theo một tiêu chí mặc định một cách dễ dàng thông qua Collections.sort().

Câu 2: Trong các lớp đó, bạn nên triển khai phương thức compareTo() như thế nào để phản ánh thứ tự sắp xếp mong muốn?
Trả lời: Chúng ta sẽ override phương thức compareTo(Media other) trong lớp Media. Để so sánh theo tiêu chí mặc định (ví dụ sắp xếp theo Title A-Z, nếu trùng Title thì sắp xếp theo Cost giảm dần), ta triển khai như sau:
@Override
public int compareTo(Media other) {
    int titleCompare = this.getTitle().compareTo(other.getTitle());
    if (titleCompare != 0) {
        return titleCompare;
    }
    return Float.compare(other.getCost(), this.getCost());
}

Câu 3: Chúng ta có thể có hai quy tắc sắp xếp (theo tiêu đề rồi đến giá, và theo giá rồi đến tiêu đề) nếu sử dụng cách tiếp cận interface Comparable không?
Trả lời: Không. Interface Comparable chỉ cho phép định nghĩa một thứ tự sắp xếp tự nhiên (natural ordering) duy nhất cho mỗi lớp thông qua phương thức compareTo(). Nếu muốn hỗ trợ nhiều quy tắc sắp xếp linh hoạt khác nhau (như sắp xếp theo các thuộc tính khác nhau tùy theo lựa chọn của người dùng tại runtime), chúng ta bắt buộc phải sử dụng interface Comparator thay vì Comparable.

Câu 4: Giả sử các DVD có quy tắc sắp xếp khác với các loại media khác, cụ thể là theo tiêu đề, sau đó giảm dần độ dài, sau đó là giá. Bạn sẽ sửa đổi mã nguồn của mình như thế nào để cho phép điều này?
Trả lời: Để cho phép điều này, trong phương thức compareTo(Media other) của lớp cha Media, chúng ta cần kiểm tra xem cả hai đối tượng so sánh có phải là thực thể của lớp DigitalVideoDisc hay không bằng cách sử dụng toán tử instanceof:
@Override
public int compareTo(Media other) {
    if (this instanceof DigitalVideoDisc && other instanceof DigitalVideoDisc) {
        DigitalVideoDisc dvd1 = (DigitalVideoDisc) this;
        DigitalVideoDisc dvd2 = (DigitalVideoDisc) other;
        
        int titleCompare = dvd1.getTitle().compareTo(dvd2.getTitle());
        if (titleCompare != 0) return titleCompare;
        
        int lengthCompare = Integer.compare(dvd2.getLength(), dvd1.getLength()); // Giảm dần độ dài
        if (lengthCompare != 0) return lengthCompare;
        
        return Float.compare(dvd1.getCost(), dvd2.getCost()); // So sánh giá
    }
    // Đối với các loại Media khác, áp dụng quy tắc mặc định thông thường
    int titleCompare = this.getTitle().compareTo(other.getTitle());
    if (titleCompare != 0) return titleCompare;
    return Float.compare(other.getCost(), this.getCost());
}
